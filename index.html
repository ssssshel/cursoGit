CURSO GIT

COMANDOS

  1. git version

    1. Versión de git

  2. git help

    1. Ayuda sobre los comandos

  3. git init

    1. Inicia un nuevo repo y crea la carpeta oculta git

  4. git status

    1. Ver que archivos han sido modificados

      • 4.1. git status -s

        1. Solo muestra archivos modificados

      • 4.2. git status -sb

        1. Vemos información de la rama maestra

  5. git add "Nombre del archivo" / git add .

    1. Agrega archivos a un área temporal (snap)

  6. git commit -m "mensaje"

    1. Agrega archivos del área temporal al repositorio local

  7. git log --oneline

    1. Permite visualizar todos los commit realizados, incluye código y mensaje

  8. git reset

    1. Permite viajar entre commits

  9. 8.1. git reset --hard "codigo de commit"

    1. Viajamos al commit especificado y borramos los cambios

  10. 8.2 git reset --hard

    1. Viajamos al último commit y borramos los cambios

  11. 8.3. git reset --mixed "codigo de commit"

    1. Viajamos al commit específico sin borrar los cambios

  12. 8.4. git reflog

    1. Visualiza todos los commit, incluyendo los que hayan sido borrados
