CURSO GIT

COMANDOS

  1. git version

    1. Versión de git

  2. git help

    1. Ayuda sobre los comandos

  3. git init

    1. Inicia un nuevo repo y crea la carpeta oculta git

  4. git status

    1. Ver que archivos han sido modificados

      • 4.1. git status -s

        1. Solo muestra archivos modificados

      • 4.2. git status -sb

        1. Vemos información de la rama maestra

  5. git add "Nombre del archivo" / git add .

    • Agrega archivos a un área temporal (snap)

  6. git commit -m "mensaje"

    • Agrega archivos del área temporal al repositorio local

  7. git log --oneline

    • Permite visualizar todos los commit realizados, incluye código y mensaje

  8. git reset

    • Permite viajar entre commits

    • 8.1. git reset --hard "codigo de commit"

      • Viajamos al commit especificado y borramos los cambios

    • 8.2 git reset --hard

      • Viajamos al último commit y borramos los cambios

    • 8.3. git reset --mixed "codigo de commit"

      • Viajamos al commit específico sin borrar los cambios

    • 8.4. git reflog

      • Visualiza todos los commit, incluyendo los que hayan sido borrados

  9. git push

    • Carga commits locales a la nube

  10. git pull

    • Descarga commits de la nube de manera local

  11. git tag

    • Enlista los tags creados

    • 11.1. git tag versionAlpha - "versionAlpha"

    • Crea un tag con un nombre y etiqueta

    • 11.2. git tag -d nombreTag

    • Elimina un tag

    • 11.3. git tag -a nombreTag f52f3da -m "versionAlpha"

    • Crea una versión en un commit anterior

    • 11.4. git show nombreTag

    • Muestra información del tag

    • 11.5. git push --tags

    • Carga los tags a la nube

  12. git branch

    • Muestra las ramas creadas y en cual de todas nos ubicamos.

    • 12.1. git branch nombreRama

    • Crea una nueva rama

    • 12.2. git checkout nombreRama

    • Cambia de una rama a otra

    • 12.3. git merge nombreRama

    • Mueve los elementos de una rama a otra

    • git branch -d nombreRama

    • Elimina una rama
  13. solucionando errores
